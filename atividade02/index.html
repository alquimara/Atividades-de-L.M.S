Introdução

Este presente artigo tem como propósito descrever o desenvolvimento do projeto Progster. O objetivo deste jogo é o ensino de lógica de programação para crianças entre sete e nove anos, de forma lúdica, por meio das teorias de ensino de Jean Piaget e de Seymour Papert.
